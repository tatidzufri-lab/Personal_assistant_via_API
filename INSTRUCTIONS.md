# Инструкция по использованию бота

Эта инструкция для вас — эксперта, который будет создавать контент для ВКонтакте с помощью бота. Никаких технических знаний не нужно.

## Шаг 1. Начало работы

Откройте бота в Telegram и нажмите **Start** (или отправьте команду `/start`). Появится главное меню с кнопками.

## Шаг 2. Настройте бота под себя (один раз)

Отправьте команду `/setup`. Бот задаст 5 вопросов:

1. Как вас зовут и в какой нише вы работаете
2. Кто ваша целевая аудитория
3. Каким тоном вы общаетесь (на ты/на вы, с юмором/строго, с эмодзи/без)
4. Какие продукты и офферы вы продвигаете
5. Особенности подачи: любимые форматы и чего в текстах быть не должно

Отвечайте своими словами, как рассказали бы живому копирайтеру. Чем подробнее ответы — тем точнее бот попадёт в ваш стиль. После этого все тексты будут написаны «вашим голосом».

Профиль можно в любой момент посмотреть командой `/profile`, а перенастроить — снова через `/setup`.

## Шаг 3. Выберите, что создать

В меню (`/menu`) семь кнопок:

| Кнопка | Когда использовать |
|---|---|
| 💡 Идеи для постов | Не знаете, о чём писать — бот предложит 10 идей под вашу задачу |
| 📝 Готовый пост | Нужна публикация на конкретную тему |
| 🔥 Прогрев | Готовите аудиторию к запуску или продаже |
| 💰 Продающий пост | Продаёте продукт, услугу или консультацию |
| 🏆 Кейс | Хотите оформить результат клиента в историю |
| 🛡 Отработка возражений | Аудитория сомневается: «дорого», «не получится», «не сейчас» |
| ✍️ Доработать мой текст | У вас есть черновик — бот усилит его и объяснит правки |

Нажмите кнопку, ответьте на вопрос бота одним сообщением — и через несколько секунд получите готовый текст с заголовком, структурой, призывом к действию и хештегами.

## Шаг 4. Доработайте результат

Не понравился вариант? Есть два способа:

- Нажмите **🔁 Ещё вариант** — бот напишет тот же текст с другой подачей.
- Напишите правку обычным сообщением: «сделай короче», «добавь юмора», «убери эмодзи», «замени призыв на запись в личные сообщения». Бот помнит контекст и внесёт изменения.

Готовый текст просто скопируйте и вставьте в пост ВКонтакте — он уже оформлен под ВК.

## Переключение нейросети

По умолчанию бот работает на OpenAI GPT-4o. Если хотите попробовать GigaChat — нажмите кнопку **🧠 Модель** в меню (или команду `/model`) и выберите нейросеть. Все ваши настройки при этом сохраняются.

## Полезные команды

- `/start` — приветствие и главное меню
- `/menu` — открыть меню сценариев
- `/setup` — настроить бота под свой стиль
- `/profile` — посмотреть, изменить или сбросить профиль
- `/model` — переключить нейросеть (OpenAI ↔ GigaChat)
- `/reset` — начать с чистого листа (очистить контекст диалога)
- `/help` — краткая справка

## Советы для лучших результатов

1. **Заполните профиль подробно.** Разница между «психолог» и «психолог, работаю с выгоранием у предпринимателей, говорю на ты, без эзотерики» — огромная.
2. **Давайте фактуру.** Для кейса укажите цифры и сроки, для продающего поста — цену и условия. Бот не выдумывает факты: там, где данных не хватило, он оставит пометку в [квадратных скобках] — заполните её перед публикацией.
3. **Дорабатывайте диалогом.** Первый вариант — черновик. Одна-две правки сообщением обычно доводят текст до идеала.
4. **Проверяйте текст перед публикацией.** Бот — помощник, но финальное слово за вами.
